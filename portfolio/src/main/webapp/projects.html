Omeka Bootswatch Theme

The omeka Bootswatch theme ports Bootswatch skins/themes for usage in Omeka. It includes various skins from Bootswatch and many opportunities to configure and customize the theme. It is a porting of Bootswatch skins originally built for Omeka v2.4 and Twitter Bootstrap 3.3.7, upgraded to target Omeka v2.6.1 and Boostrap v4.2.1.

  • Able to change layout of landing page. There are 4 options
  • Able to configure the position of the sidebar
  • Have the option to configure using bootswatch theme, there are over 20 options.

OmekaFootnotesJS

An Omeka plugin to add interactive JavaScript footnotes to exhibit pages and simple pages by extending the TinyMCE wysiwyg HTML editor.

  • Able to add, edit or remove footnotes.
  • Able to reorder footnotes
